User Health & Retention Intelligence System for Startups
Overview

This project builds a product analytics intelligence system that helps startups understand:

Why users churn
What behaviors drive retention
Where to take action for growth

Instead of basic analysis, this system transforms raw product data into actionable insights, user health scoring, and decision-ready dashboards.

Problem Statement

Startups often struggle to answer critical questions:

Which users are at risk of churn?
How does onboarding impact retention?
What behaviors indicate long-term product adoption?

This project addresses these challenges using behavioral analytics, feature engineering, and machine learning.

Dataset

A realistic SaaS dataset simulating real-world product usage:

7,000 users
220,000+ event logs
70+ engineered features
Includes:
User lifecycle data (signup, activation, churn)
Event-level product interactions
Behavioral metrics (sessions, stickiness, feature usage)
Monetization signals (MRR, upgrades)
Tech Stack
Python (Pandas, NumPy) – Data processing
Scikit-learn – Machine learning
Tableau – Dashboard visualization
CSV Pipelines – Data transformation

Dashboard System (Tableau)
Dashboard 1: User Health & Retention Command Center

Focus: Executive overview

Total Users, Churn Rate, Revenue
Action Segmentation
High-value users at risk
Dashboard 2: Time-to-Value & Activation Intelligence

Focus: Onboarding effectiveness

Key Insights:

Users taking more than 24 hours to reach value show higher churn
Early engagement strongly predicts retention
Delayed activation indicates onboarding friction

Business Action:
Improve onboarding to reduce time-to-value below 24 hours

Dashboard 3: Behavior & Engagement Intelligence

Focus: Product adoption and habit formation

Key Insights:

Higher stickiness and feature usage lead to better retention
Shallow engagement is the strongest churn signal
Multi-feature users show stronger product adoption

Business Action:
Improve feature discovery and repeat usage

Dashboard 4: Risk & Intervention Intelligence

Focus: Decision-making

Identifies users needing immediate intervention
Highlights high-value users at risk
Enables targeted retention strategies
Key Insights
Faster time-to-value reduces churn significantly
Early engagement (first 48 hours) is critical
Feature adoption depth drives retention
Friction without recovery leads to drop-off
Behavioral scoring enables actionable segmentation
Business Impact

This system enables:

Early identification of churn risk
Prioritization of high-value users
Optimization of onboarding experience
Data-driven growth decisions
